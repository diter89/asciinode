# Examples

This directory contains various examples demonstrating the capabilities of the asciinode library.

## Basic Examples

- **`basic_example.py`** - Simple diagram with LLM integration showing system architecture
- **`main_demo.py`** - Comprehensive demo showcasing various diagram features
- **`simple_grid.py`** - Basic grid layout demonstration

## Advanced Features

- **`advanced_tree_connections.py`** - Complex tree structures with rich styling and connections
- **`diagram_diff_example.py`** - Comparing two diagrams and showing differences
- **`diagram_subtree_example.py`** - Rendering specific subtrees from larger diagrams
- **`diagram_validate_example.py`** - Diagram validation and error checking
- **`diagram_titles_example.py`** - Using titles and advanced node styling

## Layout Examples

- **`omnichannel_architecture_diagram.py`** - Complex enterprise architecture with intersections
- **`test_grid.py`** - Grid layout with multiple rows and columns
- **`tree_test.py`** - Various tree layout patterns

## LLM Integration

- **`live_llm_report.py`** - Real-time LLM responses with live updates
- **`live_llm_report_01.py`** - Alternative LLM integration pattern
- **`static_llm_report.py`** - Static LLM response demonstration

## Workflow & Automation

- **`pumpfun_agent_workflow.py`** - Complete agent workflow with error handling and visual feedback
- **`automation_flow.py`** - Animated workflow demonstration
- **`agent_lifecycle.py`** - Agent lifecycle management visualization

## Specialized Examples

- **`monitoring.py`** - System monitoring dashboard
- **`neofetch.py`** - System information display (neofetch-style)
- **`crypto_map.py`** - Cryptocurrency data visualization
- **`profiled_scanner.py`** - Performance profiling visualization
- **`dobby_diagram_generator.py`** - Dynamic diagram generation

## Test Files

- **`test_basic.py`** - Basic functionality tests
- **`pumpfuntest01.py`** - Integration testing
- **`sample_diff_target.py`** - Sample file for diff testing

## Running Examples

Most examples can be run directly:

```bash
cd examples
python3 basic_example.py
python3 omnichannel_architecture_diagram.py
python3 pumpfun_agent_workflow.py "analyze system performance"
```

Some examples may require additional dependencies or environment variables. Check individual files for specific requirements.
